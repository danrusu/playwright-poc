# playwright-poc

Plywright usage POC

## 1. Prerequisites

### 1.1. [Node.js v22+](https://nodejs.org/en/download)

```bash
# check node and npm versions
node --versionnode --v
# v22.17.0
npm --version
# 10.8.2
```

### 1.2. [VS Code](https://code.visualstudio.com/download)

### 1.3. [Playwright](https://playwright.dev/)

## 2. Project setup

```bash
# install
npm install

# format code (prettier)
npm run format

# build (tsc)
npm run build
```

## 3. Execute tests

```bash
npm run test:demo
```

## 5. Reports

### 5.1. View html report

```bash
npm run report
```
